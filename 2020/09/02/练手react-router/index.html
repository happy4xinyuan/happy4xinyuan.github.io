嵌套路由

没什么难的, 复习一下. 路由的核心:

  1. 网页的入口: NavLink to='/home/news'

  2. 需要动态显示的部分: <Switch></Switch>

  3. 网页路径 与 显示部件的映射关系: <Route path='/home/message' component={Message}/>
